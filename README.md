# LGPD - Análise de Riscos (HealthTech)

Projeto acadêmico simulando consultoria de segurança da informação em uma startup de saúde.

## Contexto
A HealthTech é uma startup fictícia que desenvolve um sistema de prontuários eletrônicos integrado a serviços de saúde.

## Principais riscos
- Falta de criptografia de dados
- SQL Injection
- Falhas de controle de acesso
- Vazamento de dados sensíveis

## Soluções propostas
- TLS 1.3
- AES-256
- Prepared Statements
- WAF
- RBAC
- DLP
- OneTrust

## Objetivo
Aplicar LGPD e segurança da informação em cenário realista.
